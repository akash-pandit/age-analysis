# Notebooks

Exploratory analysis with jupyter notebooks. For reproducibility's sake, concrete workflows should be moved to python/bash scripts or a pipeline if it's a bit more involved.

- `cluster-poscablo.ipynb`: Visualized pre-clustered and re-clustered HSC + MKP populations from the lab's preprocessed Poscablo 2024 dataset (`data/rna_annotation_normalized.h5ad`)
- `cluster-rna-integrated.ipynb`: Initial integration and visualization of Poscablo + PNE (`data/pne_bonemarrow.h5ad`) datasets
- `cluster-age-integrated.ipynb`: Age-specific integration & visualization of Poscablo + PNE datasets
